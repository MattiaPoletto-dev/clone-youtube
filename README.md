# Clone YouTube
## Sobre o projeto
Posso considerar esse projeto uma pequena conquista, mas o conhecimento para ser um bom dev, ao meu ver, vem aos poucos. Errei, errei e errei mais uma vez até acertar, se eu travasse em determinado problema (e após tentar bastante), apelava para IA, mas mesmo assim, o importante era entender o que tinha me dado de solução para determinado problema, e próxima vez que eu me deparasse com algo parecido, conseguisse resolver por mim mesmo.
## Objetivo
Reproduzir uma cópia da página inicial do youtube, focando em HTML e CSS puro (sem a utilização de frameworks).
## Aprendizados
Foram bastantes, algumas coisas já conhecia, outras precisei recordar e outras aprendi, mas as principais foram:
- Positions: relative, absolute, fixed, sticky;
- Displays: flexbox, grid;
- Media queries para responsividade;
- Seletores: id, classes, pseudoclasses;
Obs: esses foram os principais aprendizados
