# Clone página inicial do YouTube
## Sobre o projeto
Posso considerar esse projeto uma pequena conquista, mas o sucesso para ser um bom dev, ao meu ver, vem de pouco em pouco. Errei, errei e errei mais uma vez até acertar, se eu travasse em determinado problema (e após tentar bastante) apelava para IA, mas mesmo assim, o importante para mim era entender o que ela tinha me dado de solução para determinado problema, e próxima vez que eu me deparasse com um algo parecido, conseguisse resolver por conta própria.
## Objetivo
Reproduzir uma cópia da página inicial do youtube, focando em HTML5 e CSS3 puro (sem a utilização de frameworks).
## Aprendizados
Foram bastantes, algumas coisas já conhecia, outras precisei recordar e outras aprendi do zero, mas as principais e de maior valor foram:
- Positions: relative, absolute, fixed, sticky;
- Displays: flexbox, grid;
- Media queries para responsividade;
- Seletores: id, classes, pseudoclasses;
- Muitas outras propriedades do CSS.
